================================================================================
                            iPod Wrapped - Portable
                         Windows Installation Guide
================================================================================

Thank you for downloading iPod Wrapped!

This is a PORTABLE application - no installation required!

================================================================================
GETTING STARTED
================================================================================

1. Extract this entire folder to any location on your computer
   - Desktop, Documents, USB drive - anywhere works!

2. Double-click "iPodWrapped.exe" to run the application

3. Follow the on-screen instructions to set up your Last.fm API credentials

That's it! The app is ready to use.

================================================================================
REQUIREMENTS
================================================================================

- An iPod running Rockbox with logging enabled
- Last.fm API credentials (free - get them at https://www.last.fm/api)
- Windows 10 or Windows 11

================================================================================
YOUR DATA
================================================================================

All your data is stored in the "storage/" folder inside this directory:
- Database files
- Album art
- Settings
- Credentials (securely stored using Windows Credential Manager)

This means you can:
- Move the entire folder anywhere
- Copy it to a USB drive
- Backup by copying the whole folder

================================================================================
UPDATING
================================================================================

To update to a new version:

1. Download the new version ZIP
2. Extract it to the SAME location as your current installation
3. When asked, choose to REPLACE/OVERWRITE the existing files
4. Your data in "storage/" will be preserved automatically

================================================================================
UNINSTALLING
================================================================================

Simply delete this entire folder. That's it!

Your credentials stored in Windows Credential Manager can be removed via:
Control Panel > Credential Manager > Windows Credentials

================================================================================
TROUBLESHOOTING
================================================================================

If the app doesn't start:
- Make sure you extracted ALL files from the ZIP
- Try running as Administrator (right-click > Run as administrator)
- Check Windows Defender hasn't quarantined the .exe file

If you encounter issues:
- Report them at: https://github.com/Mandy-cyber/Everything-iPod/issues
- Include your Windows version and any error messages
